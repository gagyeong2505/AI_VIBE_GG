## CSS 변수 정의

```css
:root {
  /* Primary Colors */
  --color-primary: #7ED84B;         /* 라임 그린 — 배경, 메인 색조 */
  --color-primary-dark: #5AB832;    /* 딥 라임 그린 — 호버, 강조 섹션 */

  /* Accent Colors */
  --color-accent: #E8583A;          /* 코랄 레드 — 제목, 일러스트, 포인트 텍스트 */
  --color-accent-light: #F07055;    /* 라이트 코랄 — 링크 호버, 부드러운 강조 */

  /* Neutral */
  --color-white: #FFFFFF;           /* 퓨어 화이트 — 메인 레터링, 카드 배경 */

  /* Illustration Colors */
  --color-peach-light: #F5A882;     /* 소프트 피치 — 복숭아 과육 밝은 영역 */
  --color-peach-dark: #D97B5A;      /* 웜 피치 다크 — 복숭아 과육 음영 */

  /* Background */
  --color-bg-primary: #7ED84B;      /* 포스터 기준: 라임 그린 메인 배경 */
  --color-bg-secondary: #FFFFFF;    /* 카드, 모달 배경 */

  /* Text */
  --color-text-on-green: #FFFFFF;   /* 라임 그린 위 텍스트 */
  --color-text-accent: #E8583A;     /* 강조 텍스트, 섹션 제목 */

  /* Border */
  --color-border: rgba(232, 88, 58, 0.3); /* 코랄 계열 테두리 */
}
```

## 컬러 사용 가이드

### Primary — `#7ED84B` (라임 그린)
- 페이지 또는 섹션 전체 배경 (dominant)
- 뱃지, 태그 배경
- 아이콘 fill 색상
- **사용 비율 목표**: 전체 화면의 55~65%

### Primary Dark — `#5AB832` (딥 라임 그린)
- 버튼 호버·포커스 상태
- 진한 구분선 또는 카드 테두리 강조
- **사용 비율 목표**: 전체 화면의 5% 이하

### Accent — `#E8583A` (코랄 레드)
- 헤더 제목, 이벤트 날짜·장소 텍스트
- 포인트 일러스트 아웃라인
- CTA 버튼 텍스트 또는 아이콘
- **사용 비율 목표**: 전체 화면의 10~15%

### Accent Light — `#F07055` (라이트 코랄)
- 링크 호버 효과
- 서브 캡션, 보조 설명 텍스트
- **사용 비율 목표**: 포인트 사용 (5% 이하)

### White — `#FFFFFF`
- 배경(라임 그린) 위에 올라오는 대형 레터링
- 카드, 모달 배경
- 버튼 텍스트 (코랄 레드 버튼 위)
- **사용 비율 목표**: 전체 화면의 20~30%

### Illustration — `#F5A882` / `#D97B5A` (피치 계열)
- 복숭아·과일·자연물 일러스트 fill
- 아바타, 썸네일 일러스트 전용
- **주의**: UI 기능 요소에는 사용하지 않음 — 일러스트 한정

```
서브텍스트: #E8583A (코랄 레드)
```

```
배경: #E8583A (코랄 레드)
텍스트: #FFFFFF (화이트)
호버 배경: #C94A2A (10% 어둡게)
```

```
테두리: #E8583A (코랄 레드)
```

## 반응형 고려사항

### 다크모드 대응

```css
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-primary: #2A5A18;          /* 라임 그린 → 딥 포레스트 그린 */
    --color-bg-secondary: #1C3A10;
    --color-primary: #7ED84B;             /* 라임은 어두운 배경에서도 선명하게 유지 */
    --color-accent: #F07055;              /* 코랄 밝기 약간 상향 */
    --color-text-on-green: #F0F0F0;
    --color-border: rgba(232, 88, 58, 0.4);
  }
}
```

### 접근성 (Accessibility)

| 조합 | 대비율 (추정) | WCAG 등급 |
|------|--------------|-----------|
| `#FFFFFF` on `#7ED84B` | 3.2:1 | A ⚠️ (Large text 한정) |
| `#E8583A` on `#7ED84B` | 2.1:1 | ✗ (장식용·일러스트 전용) |
| `#E8583A` on `#FFFFFF` | 4.5:1 | AA ✅ |
| `#FFFFFF` on `#E8583A` | 4.5:1 | AA ✅ |
| `#1A1A1A` on `#FFFFFF` | 18.1:1 | AAA ✅ |

> ⚠️ 라임 그린(`#7ED84B`) 위 화이트 텍스트는 대비율이 낮으므로 **본문 소형 텍스트에는 사용 금지**.
> 포스터처럼 대형 레터링·장식 용도로만 사용 권장.

### 모바일 (xs: 0~599px)
- 버튼 최소 터치 영역: `48px × 48px` 이상
- 라임 그린 배경 위 텍스트는 반드시 굵기 `font-weight: 700` 이상으로 가독성 확보
- 코랄 레드 포인트는 1개 섹션당 1곳으로 제한

### 데스크톱 (md: 900px~)
- 라임 그린 배너 full-width 적용 가능
- 피치 계열 일러스트 요소 여백에 배치하여 시각적 생동감 부여
