## 컬러 사용 가이드

### Primary — `#E63012` (레드-오렌지)
- 전체 페이지 배경 또는 히어로 섹션 배경
- 강렬한 인상을 주어야 하는 풀블리드(full-bleed) 섹션
- 식욕·긴장감·에너지를 강조하고 싶은 F&B / 이벤트 배너
- **사용 비율 목표**: 전체 화면의 55~65%

### Primary Dark — `#C42508` (다크 레드)
- 호버 시 배경 어두워지는 효과
- 섹션 하단 그라데이션, 그림자 처리
- 배경과 콘텐츠의 깊이감 연출
- **사용 비율 목표**: 전체 화면의 5~10% (섀도우·구분 용도)

### Secondary — `#F2E8A0` (크리미 옐로우)
- 헤드라인 타이포그래피 (레드 배경 위 최고 대비)
- 인용 부호, 해시태그 기호 등 그래픽 요소
- CTA 버튼 배경 (레드 배경 위에서 시선 집중 효과 극대화)
- 카드·배지 테두리
- **사용 비율 목표**: 전체 화면의 15~20%

### Accent — `#D4414A` (레드-핑크)
- 포인트 장식 테두리 (컵 림, 카드 하이라이트 라인)
- 뱃지, 태그 등 소형 강조 요소
- **주의**: 레드 배경 위에서 명도 차이가 적어 텍스트에는 사용 지양

### Text — `#F2E8A0` / `#FFFFFF` / `#F5D9D9`
- `#F2E8A0`: H1~H2 헤드라인 — 레드 배경 위 최우선 가독성 확보
- `#FFFFFF`: 본문, 서브 카피, 주소·연락처 등 상세 정보
- `#F5D9D9`: 날짜, 태그, 부가 메타 정보 (눈에 띄지 않아야 할 요소)

### Interactive — 버튼 & 링크
- 기본 CTA 버튼: 배경 `#F2E8A0` + 텍스트 `#E63012` (레드 배경 위 최고 대비)
- 호버 시: 배경 `#EDE090` (크리미 옐로우 10% 어둡게)
- 아웃라인 버튼: 테두리 `#FFFFFF` + 텍스트 `#FFFFFF`
- 링크 텍스트: `#FFFFFF` → 호버 시 `#F2E8A0`으로 전환

## 반응형 고려사항

### 다크모드 대응

```css
@media (prefers-color-scheme: dark) {
  :root {
    /* 레드-오렌지 계열은 다크모드에서도 그대로 유지 — 브랜드 아이덴티티 보존 */
    --color-primary: #E63012;
    --color-primary-dark: #A81E05;   /* 다크모드에서 더 깊은 레드 */

    /* 배경은 레드 계열 유지하되 채도를 약간 낮춤 */
    --color-bg-primary: #CC2A0F;
    --color-bg-secondary: #A81E05;

    /* 텍스트는 밝기 유지 */
    --color-text-primary: #F2E8A0;
    --color-text-secondary: #FFFFFF;
    --color-text-muted: #E8C8C8;

    --color-border: #F2E8A0;
    --color-border-accent: #D4414A;
  }
}
```

### 모바일 (xs: 0~599px)
- 히어로 배경 `#E63012` 전면 적용, 텍스트 크기 최소 `16px` 유지
- CTA 버튼 최소 터치 영역: `48px × 48px` 이상
- WCAG AA 기준: `#F2E8A0` on `#E63012` 대비율 **3.8:1** → 대형 텍스트(18px+)에 한해 AA 충족
- 소형 텍스트는 반드시 `#FFFFFF` 사용 (대비율 **4.6:1** — AA ✅)

### 태블릿 (sm: 600~899px)
- 배경 레드 유지, 콘텐츠 좌우 패딩 `px: 3` 이상
- 카드 테두리 `--color-border` 1px 유지
- 버튼 폰트 사이즈 `1rem` → `1.1rem` 확대

### 데스크톱 (md: 900px~)
- 풀블리드 히어로 섹션에서 레드 배경의 임팩트 극대화
- 호버 인터랙션 적극 활용: `--color-button-hover`, `--color-link-hover`
- 헤드라인 크기 `clamp(3rem, 8vw, 7rem)` 적용으로 포스터 감성 유지

### 접근성 (Accessibility)

| 조합 | 대비율 | WCAG 등급 |
|------|--------|-----------|
| `#FFFFFF` on `#E63012` | 4.6:1 | AA ✅ |
| `#F2E8A0` on `#E63012` | 3.8:1 | AA (대형 텍스트) ✅ |
| `#E63012` on `#F2E8A0` | 3.8:1 | AA (대형 텍스트) ✅ |
| `#FFFFFF` on `#C42508` | 6.1:1 | AA ✅ |
| `#F2E8A0` on `#C42508` | 5.0:1 | AA ✅ |

> ⚠️ `#F2E8A0` on `#E63012` 조합은 소형(14px 이하) 본문 텍스트에는 사용하지 않는다.
> 소형 텍스트는 반드시 `#FFFFFF`를 사용하여 가독성을 확보한다.
